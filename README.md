# 庫存管理系統 (Inventory Management System)

這是一個使用 Python 開發的輕量化庫存管理工具，旨在幫助使用者高效追蹤與管理零件庫存。

## 🌟 主要功能
* **自動載入資料**：啟動時自動讀取 parts.txt (或 CSV) 檔案。
* **即時查詢**：透過零件編號或名稱快速搜尋。
* **庫存更新**：直覺式的庫存增減介面。
* **異常處理**：針對檔案遺失或輸入錯誤設有完善的錯誤提示。

## 🛠️ 使用技術
* **語言**：Python 3.12
* **版本控管**：Git / GitHub

## 🚀 如何執行
1. 確保電腦已安裝 Python。
2. 下載此專案所有檔案。
3. 在終端機執行：
   ```bash
   python InventoryManager.py
